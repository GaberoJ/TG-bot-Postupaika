# Telegram-бот для профориентации

Бот помогает абитуриентам подбирать направления и программы обучения в российских вузах, используя базу данных, собранную с сайта **postupi.online**, и рекомендации на основе интересов пользователя с помощью **Yandex GPT**.

---

## Возможности

- Интерактивный диалог с ИИ-ассистентом для формирования списка направлений по вашим предпочтениям (предметы ЕГЭ, увлечения).
- Просмотр информации о направлении: код, количество программ, вузов, городов, бюджетных мест и минимальная стоимость.
- Поиск вузов по направлению и городу.
- Детальный просмотр программ обучения: форма обучения, уровень, бюджетные/контрактные места, проходные баллы, стоимость, описание.
- Топ-10 программ с самым высоким и самым низким проходным баллом.
- Сохранение понравившихся направлений и программ в личный список.
- Удобная навигация через кнопки и меню.

---

## Технологии

- **Python 3.12**
- **Aiogram 3** — асинхронный фреймворк для Telegram Bot API
- **OpenAI-совместимый клиент** для Yandex Cloud Assistant
- **PostgreSQL 16** — основная база данных
- **BeautifulSoup4 + requests** — парсинг данных
- **Docker & Docker Compose** — контейнеризация
- **Многопоточность** — для ускорения массового парсинга
